Set up the framework in your Vue application using the headless context pattern and peripheral integration packages.

## Installation

The framework requires [@tanstack/vue-query](https://www.npmjs.com/package/@tanstack/vue-query) to manage data fetching, caching, and synchronization.

::code-group

```bash [pnpm]
pnpm add @ginjou/vue @tanstack/vue-query
```

```bash [yarn]
yarn add @ginjou/vue @tanstack/vue-query
```

```bash [npm]
npm install @ginjou/vue @tanstack/vue-query
```

```bash [bun]
bun add @ginjou/vue @tanstack/vue-query
```

::

## Provider Setup

The framework uses a **Context API** pattern to provide configuration to your application. Instead of a single monolithic plugin, you use granular `define*Context` functions within the `setup` block of your root component.

Available setup functions:

- `defineFetchersContext`: Provide your data fetchers.
- `defineQueryClientContext`: Provide the TanStack Query client.
- `defineResourceContext`: Provide resource definitions and patterns.
- `defineRouterContext`: Provide the router integration.
- `defineAuthContext`: Provide the authentication provider.
- `defineAuthzContext`: Provide the authorization provider.
- `defineI18nContext`: Provide the internationalization provider.
- `defineNotificationContext`: Provide the notification provider.

## Peripheral Services

Use integration packages to bridge your favorite libraries with the framework.

### Router

Use `@ginjou/with-vue-router` to connect your application with `vue-router`.

```typescript
import { createRouter } from '@ginjou/with-vue-router'
import { defineRouterContext } from '@ginjou/vue'

// In setup()
defineRouterContext(createRouter())
```

### I18n

Use `@ginjou/with-vue-i18n` to connect with `vue-i18n`.

```typescript
import { createI18n } from '@ginjou/with-vue-i18n'
import { defineI18nContext } from '@ginjou/vue'

// In setup()
defineI18nContext(createI18n())
```

### REST API

Use `@ginjou/with-rest-api` to quickly connect to any RESTful backend.

```typescript
import { createFetcher } from '@ginjou/with-rest-api'
import { defineFetchersContext } from '@ginjou/vue'

// In setup()
defineFetchersContext({
  default: createFetcher({ url: 'https://api.example.com' })
})
```

## Complete Setup Example

Here is a full example of a root component (`App.vue`) configured with common providers and integrations.

```vue [App.vue]
<script setup lang="ts">
import { defineFetchersContext, defineQueryClientContext, defineResourceContext, defineRouterContext } from '@ginjou/vue'
import { createFetcher } from '@ginjou/with-rest-api'
import { createRouter } from '@ginjou/with-vue-router'
import { QueryClient } from '@tanstack/vue-query'

// 1. Initialize TanStack Query
const queryClient = new QueryClient()

// 2. Define Resources
const resources = [
  {
    name: 'posts',
    list: '/posts',
    create: '/posts/new',
    edit: '/posts/:id/edit',
    show: '/posts/:id',
  },
]

// 3. Provide context
defineQueryClientContext(queryClient)
defineRouterContext(createRouter())
defineFetchersContext({
  default: createFetcher({ url: 'https://api.example.com' }),
})
defineResourceContext({ resources })
</script>

<template>
  <RouterView />
</template>
```
