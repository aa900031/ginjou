# ColorPicker

A component to select a color.

## Key Props

- `format`: to set `rgb` value of the ColorPicker.
- `throttle`: to set the throttle value of the ColorPicker.
- `size`: to set the size of the ColorPicker.
- `disabled`: to disable the ColorPicker.
