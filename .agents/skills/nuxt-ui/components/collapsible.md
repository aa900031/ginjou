# Collapsible

A collapsible element to toggle visibility of its content.

> Based on [Reka UI Collapsible](https://reka-ui.com/docs/components/collapsible)

## Key Props

- `disabled`: to disable the Collapsible.

## Usage

```vue
<UCollapsible
  <!-- props here --
>
/>
```

## Slots

- `#content`
